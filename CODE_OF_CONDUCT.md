# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

For the full text, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported to the project maintainers at **conduct@merkleworks.com**.

All complaints will be reviewed and investigated promptly and fairly. Maintainers are obligated to respect the privacy and security of the reporter.

## Scope

This Code of Conduct applies within all project spaces, including issues, pull requests, discussions, and any public space where an individual is representing the project.
